# stolink
**작가를 위한 AI 기반 스토리 관리 플랫폼** <br>
- 장편 소설 집필에 필요한 복선 관리, 캐릭터 관계도, 세계관 설정, 일관성 체크를 지원하는 통합 집필 환경입니다.

### 웹페이지 주소

- #### https://storead.stolink.link

### 주요 기능

- **스마트 에디터** - Tiptap 기반 리치 텍스트 에디터로 몰입감 있는 글쓰기 경험
- **복선 관리** - 이야기 전반에 걸친 복선을 태그하고 추적
- **캐릭터 관계도** - Neo4j 그래프 DB 기반 인물 관계 시각화
- **세계관 설정** - 체계적인 세계관 데이터 관리
- **AI 일관성 체크** - 스토리 내 모순 및 설정 오류 자동 검출
- **디렉토리 문서 구조** - 폴더/텍스트 재귀 구조로 유연한 원고 관리
